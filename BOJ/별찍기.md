```
for i in range(1, n+1) :
    print(" "*(n-i), "*"*i)
```

2442 별찍기_5
-------------
```
n = int(input())
h = n//2
for i in range(0, n) :
    for j in range((-2)*h, 2*h+1) :
        if (j<=i) and (-j<=i) :
            print("*", end = '')
        else :
            print(' ', end = '')
    print()
```
처음엔 이중반복문을 이렇게 사용했다가, 자꾸 출력 형식이 잘못되었다는 결과가 나옴.   
자세히 보니 출력 시, 삼각형 왼쪽엔 공백이 들어가면 안되길래 그냥 문자열 곱셈으로 해결함.
